# 1月24日

昨晚，招商银行发布了2016年业绩快报，营收增长约4%，净利润增长7%+。从今天市场反应来看，应该是超预期了。昨天卖出的海兴电力，却逆势上涨1%+，但次新股板块却下跌约2%。所以说，卖出的原因是对的，预料到了市场的回调压力，但是从结果来看，确实失败的。

# 1月23日

考虑到海兴电力的表现比较疲软：次新股指数上涨时，他的涨幅比指数小；指数下跌时，他的跌幅比指数大。并且春节前这几天，市场整体走势偏弱，因此卖出海兴电力一半仓位。
